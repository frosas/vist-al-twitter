### Usage

```bash
$ npm install
$ export VIST_AL_TWITTER_CONSUMER_KEY="..."
$ export VIST_AL_TWITTER_CONSUMER_SECRET="..."
$ export VIST_AL_TWITTER_ACCESS_TOKEN="..."
$ export VIST_AL_TWITTER_ACCESS_TOKEN_SECRET="..."
$ [DEBUG=*] bin/run
```

### Development

```bash
$ node_modules/.bin/gulp test
$ npm run lint
```
